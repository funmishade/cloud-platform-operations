# Cloud Platform Operations

## Overview
This repository represents ongoing day-to-day DevOps and platform engineering
tasks across Linux systems, AWS infrastructure, Terraform-based IaC, and
Kubernetes workloads.

The focus is on operational reliability, automation, security, and
production-readiness rather than isolated demos.

## Responsibilities Covered
- Linux system administration and troubleshooting
- AWS infrastructure design and management
- Infrastructure as Code using Terraform
- Kubernetes workload deployment and operations
- Monitoring, alerting, and resource optimization

## Key Practices
- Infrastructure modularization
- Least-privilege IAM
- Resource requests and limits
- Environment separation (dev/prod)
- Operational documentation and playbooks

## Tooling
- Linux
- AWS (EC2, VPC, IAM, EKS)
- Terraform
- Kubernetes
- Helm
- Bash scripting

## Audience
- DevOps Engineers
- Platform Engineers
- SREs
- Hiring Managers & Interviewers
